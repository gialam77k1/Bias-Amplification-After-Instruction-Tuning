    Scope and caveats

    • The demo does not claim that instruction tuning always increases bias.
    • It measures whether each model-dataset pair moves closer to or further away from benchmark-specific reference points.
    • Results depend on model size, dataset choice, benchmark design, scoring method, and training steps.
    • The current ablation file is endpoint-only, not a full learning curve.
    • Live prompt testing is qualitative and should not replace benchmark-backed conclusions.
